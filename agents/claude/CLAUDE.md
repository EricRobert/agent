# General Guidelines

## Defaults

* Simpler design wins unless proven otherwise.
* Standard library first.
* Look for and follow any project-specific AGENTS.md when working in a repository.

## Agent Behavior

* Change the smallest surface area that solves the root problem.
* Do not invent abstractions early.

## Code Rules

* Prefer explicit, idiomatic code.
* Clear, short names.
* Do not add comment to narrate the code.
